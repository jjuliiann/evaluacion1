## Origen y Descripción de los Datos
El dataset seleccionado es el **"FIFA 21 Messy Raw Dataset"** de Kaggle. Contiene estadísticas de rendimiento, datos biométricos y detalles contractuales de más de 18,000 jugadores. 
* **Variable Objetivo Sugerida:** `Value` (Valor de mercado).
* **Estado:** El archivo se encuentra en formato 'Raw', presentando inconsistencias en unidades de medida, caracteres especiales de moneda y una alta tasa de valores nulos.

## Justificación del Entorno
Hemos seleccionado **Google Colab** como entorno de desarrollo principal por las siguientes razones:
1. **Colaboración en Tiempo Real:** Permite el desarrollo simultáneo entre los integrantes del equipo.
2. **Integración con GitHub:** Facilita el versionado directo del notebook mediante la opción "Guardar copia en GitHub", manteniendo la trazabilidad que exige la pauta.
3. **Ecosistema:** Ofrece pre-instaladas todas las librerías necesarias para Pipelines (Scikit-Learn) y visualización (Seaborn/Matplotlib).
